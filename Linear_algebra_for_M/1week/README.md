## 선형대수학을 위한 벡터란?

+ 스칼라(scaler) : 크기만 가지는 단일 값 ex) 시속 5m/s
+ 벡터(vector) : 크기와 방향을 모두 표현하는 값 ex) 동쪽으로 시속 5m/s
  + 크기는 길이로 방향은 화살표로 벡터를 표현할 수 있다.
  + 글로 벡터를 표현할 때에는 (0,5), (0,1,5)와 같이 쓴다.
