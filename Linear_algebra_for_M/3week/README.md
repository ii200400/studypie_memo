## 고유값과 고유벡터란?

+ 고유벡터(eigenvector) : 선형 변환이 일어난 후에도 방향이 변하지 않는, 영벡터가 아닌 벡터
+ 고유값(eigenvalue) : 고유벡터가 변환되는 '크기'
 + 만약 변환 T에 의해 벡터 v가 스칼라 c만큼만 변한다면(T(v)=cv) c는 고유값, v는 고유벡터이다.
 + 고유벡터가 기저벡터를 포함한다..
